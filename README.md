# Privacy Policy — Profit Calculator PRO

**Last updated:** 2026-03-18

Profit Calculator PRO is a local calculator Chrome extension. This policy explains what data it uses and how.

## Summary

- The extension **does not collect, sell, or share** personal data.
- The extension **does not transmit** your calculation inputs to any server.
- The extension uses **Chrome storage** to save your profiles and to keep the popup state from resetting.

## What data the extension stores

Profit Calculator PRO stores the following data **locally in your browser** using `chrome.storage.local`:

- **Profiles** you save (by name), including:
  - VAT / tax rate
  - Your cost rows (cost name, value, amount/% type, and “when applies” setting)
  - Your “base costs” selection for “After selected costs”
- **Draft state** (auto-saved) so closing/reopening the popup keeps your last entries, including:
  - Brutto sale input
  - VAT / tax rate input
  - Current cost rows
  - Current “base costs” selection

## What data the extension does NOT collect

The extension does **not**:

- Collect browsing history
- Track websites you visit
- Record keystrokes outside the extension
- Use analytics or advertising identifiers
- Send calculation inputs, profiles, or drafts to a remote server

## Permissions

The extension requests:

- **`storage`**: to save profiles and draft state locally so the popup does not reset and you can switch between marketplace profiles.

## Data sharing

No data is shared with third parties.

## Data retention & deletion

Because data is stored locally in your browser:

- You can delete saved data by removing profiles inside the extension (Delete button).
- You can delete all stored data by uninstalling the extension or clearing extension storage in Chrome.
